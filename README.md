# Slicking
## Build Instructions
You need odin

Build with odin build . -debug

or odin run .

## Usage
### Tab Groups
Tab to cycle through tab groups
1-9 switch tools

**Group 1: Tile Editing**
1. Snow Tile
2. Dirt Tile
3. Ice Tile
4. Water Tile
5. Wall Tile

**Group 2: Thing Editing**
1. Penguine King
2. Blue Balls

**Group 3: Player Tools**
1. JETPACK (NEW!!!)
   - *only works on ice*
3. (NEXT) Snow Gun
4. (NEXT) Ice Gun
5. (NEXT) Gun Gun
6. (NEXT) Carrot Gun (for commanding the army of snowmen)
